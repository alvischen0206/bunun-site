LINE / AI Agent / n8n

LINE AI 業務助理

手機聊天情境連到後端節點流程，說明 LINE 訊息如何進入、分類、觸發 AI 回覆與回傳。

  • 01入口：LINE 訊息先被整理成可判斷的任務。
  • 02分流：n8n 依照意圖決定要查資料、問 AI 或留下人工處理。
  • 03輸出：回覆客戶，同時留下紀錄，方便後續追蹤。

LINE 公開資訊助理

把天氣、AQI 或公開資料 API 包成 LINE 可問的服務。

  • 01入口：使用者用 LINE 問天氣、空氣品質或公開資料。
  • 02整合：n8n 呼叫 API，整理欄位與格式。
  • 03輸出：用白話回覆，不讓使用者看複雜資料表。

LINE 名片掃描助理

從圖片進入 OCR，再整理聯絡資訊與確認欄位。

  • 01入口：使用者上傳名片圖片。
  • 02辨識：OCR 抽出姓名、公司、電話、Email 等欄位。
  • 03輸出：整理成可匯入、可追蹤的聯絡資料。

Tender AI 標案查詢

漏斗式查詢圖，呈現標案問題如何經過文件檢索、條件整理與 AI 回答。

  • 01入口：把 PDF、OCR 結果與標案欄位整理成可檢索資料。

  • 03輸出：回答附上來源方向，降低人工翻文件時間。
